# Reflexão Arquivos

●​ Qual a vantagem de transformar cada linha do arquivo em dicionários
em vez de trabalhar apenas com strings?
R: Os dicionarios trazem uma legibilidade e acesso dinamico, permitindo 
a criação de banco de dados organizados.

●​ Em que situações pode ser útil retornar uma tupla de registros 
(como nos exercícios ex01 e ex02) em vez de apenas uma lista de linhas?
R: A tuplas são ideias para garantir a solidez e a integridade dos 
dados.Permitindo registros seguros e imutaveis

●​ O que você achou mais desafiador: ler o arquivo ou transformar as
linhas em estruturas de dados (dicionários)?
R: Ler o arquivo atraves da função open() é a parte mais simples,
agora para estruturar os dados preciso entender como eles estão
escritos, como seram separados e organizados.
